# Tej-Analytics Portfolio Website

This is a static portfolio website designed for the GitHub Pages repository:

```text
Tej-Analytics.github.io
```

## Tech stack

- HTML5
- CSS3
- Vanilla JavaScript
- Canvas particle animation
- GitHub Pages hosting

## Files

```text
index.html
styles.css
script.js
assets/Tejas_R_Nighojkar_Resume.pdf
README.md
.nojekyll
```

## How to use

1. Open `index.html` locally to preview the site.
2. Replace or edit the content in `index.html` as needed.
3. Keep the resume file inside the `assets` folder.
4. Push all files to your `Tej-Analytics.github.io` repository.
5. Open `https://Tej-Analytics.github.io/` after GitHub Pages deploys.

## GitHub Pages setup

In GitHub:

1. Go to the repository `Tej-Analytics.github.io`.
2. Open Settings.
3. Open Pages.
4. Choose `Deploy from a branch`.
5. Select branch `main` and folder `/root`.
6. Save.

GitHub Pages may take a few minutes to publish the site.
